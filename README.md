# Kijamii

> In this project we built a social media website with the features such as creating posts, liking and sending friend requests.

## Built With

- Ruby v2.7.0
- Ruby on Rails v5.2.4

## Live Demo

[LIVE DEMO](https://evening-peak-11626.herokuapp.com/)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ruby: 2.7.0
Rails: 5.2.3
Postgres: >=9.5

### Setup

Clone the repository to your local machine:

```
git clone https://github.com/paulzay/ror-social-scaffold.git

cd ror-social-scaffold

```

Install gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
```

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    bundle exec rspec
```
